# 提案書レビュー: 社内ナレッジ共有システム導入提案

## 強み

- 課題が定量的に示されており（オンボーディング3ヶ月、週10時間のQ&A）、提案の必要性が明確
- ROIの言及があり、経営層へのアピールポイントがある
- パイロット → 全社展開のフェーズ分けがあり、リスク低減策として適切

## 懸念点

- ROI試算の根拠が示されていない（人件費削減効果の計算式が不明）
- 既存ドキュメント移行の工数見積もりが欠落している
- Notionへの定着を促す施策が「月次レビュー会」のみで薄い

## 改善提案

1. ROI試算の根拠を別紙で示す（時給単価 × 削減時間 × 人数）
2. 移行対象ドキュメント数と移行工数の概算を追記する
3. 定着化施策を強化する（例: 検索ファーストの文化醸成、ページ品質のKPI設定）

## 未解決の問題

- セキュリティ・情報漏洩リスクはどう評価するか（Notion上に機密情報を置く場合の方針）
- 既存ツール（Confluence等）の契約はいつ終了するか
- 担当者（田中）の工数はどの程度か、兼務か専任か
