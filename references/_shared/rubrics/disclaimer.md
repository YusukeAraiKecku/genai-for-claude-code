## 免責文テンプレート（汎用）

以下のいずれかを出力の末尾または冒頭に置く。

### 法的アドバイス免責
```
本スキルの出力は情報提供・整理を目的とするものであり、法的アドバイスではありません。
個別の判断は必ず資格を持つ専門家にご確認ください。
```

### 医療判断免責
```
本スキルの出力は医療上の診断・治療方針の決定に使用しないでください。
医療判断は必ず医療専門職にご相談ください。
```

### 財務・税務判断免責
```
本スキルの出力は方針整理であり、確定的な財務・税務判断ではありません。
最終判断は資格を持つ専門家の確認を要します。
```

## eval での use_include 設定例

```yaml
must_include:
  - "本スキルの出力は方針整理"
must_not_include:
  - "確定的に節税できます"
  - "弁護士に相談せずに"
```
